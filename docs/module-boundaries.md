# Limites entre core, módulos e plugins

Este documento define como classificar novas peças do ORBIT:

| Tipo | Papel |
|------|--------|
| **Core** | Fundação partilhada, contratos estáveis, poucas dependências opcionais. |
| **Módulo** | Funcionalidade de produto coesa, acoplada ao core por interfaces acordadas. |
| **Plugin** | Extensão opcional, substituível, com fronteira clara face ao core e aos módulos. |

Na Fase 0 ainda não há módulos nem plugins listados; esta tabela serve de referência para decisões futuras.
